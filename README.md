## 币安合约自动开多单和空单
3m kline, 2日 > 20日 (当前k线)买多，2日 < 20 日 卖

买入
当前k线 2日 > 20日 时，此时设置监控点，等待该k线结束
当下一根k线 开始后，连续 三次  2日 > 20日 时，以市价买入

卖出
当前k线 2日 < 20日 时，此时设置监控点，等待该k线结束
当下一根k线 开始后，连续 三次 2日 < 20日 时，以市价卖出


### install
```
pnpm install
```

### start

- init
```
node binance.js // 初始化数据库,运行 10 秒后手动停止
```

- run
```
node index.js // 开启服务
node web/app.js // 开启前端管理界面
```
